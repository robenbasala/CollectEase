1. Open https://console.firebase.google.com → your project.
2. Project settings (gear) → Service accounts.
3. Click "Generate new private key" → download the .json file.
4. Save it here as exactly: firebase-service-account.json
   (folder: backend/secrets/)

5. Restart the backend: npm run dev (from the backend folder).

User roles and property access are stored in Firebase Authentication custom claims (no Firestore).
Do not commit firebase-service-account.json — it is listed in backend/.gitignore.
